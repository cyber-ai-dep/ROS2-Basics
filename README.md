# Getting Started with ROS 2: From Basics to Manipulator Simulation

[![License](https://img.shields.io/badge/License-Apache%202.0-blue.svg)](LICENSE)
[![ROS 2 Jazzy](https://img.shields.io/badge/ROS%202-Jazzy-blue)](https://docs.ros.org/en/jazzy/)

Complete hands-on course for learning ROS 2 from scratch to robot simulation and motion planning.

## Course Overview

**Duration:** 4 weeks  
**Format:** 9 hands-on workshops  
**Level:** Beginner to Intermediate

### What You'll Learn

- Linux fundamentals and terminal usage
- Python programming for robotics
- ROS 2 communication (Topics, Services, Actions)
- Robot description with URDF
- 3D visualization with RViz2
- Physics simulation with Gazebo
- Sensor integration and data processing
- Motion planning with MoveIt2

## Getting Started

### Prerequisites
- Ubuntu 24.04 (VM or Dual Boot)
- 8GB RAM minimum
- Internet connection
- No prior ROS experience required
