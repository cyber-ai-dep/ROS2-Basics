TF Concepts

Coordinate Frames

Each frame represents a 3D coordinate system with origin point and orientation. Every robot component has its own frame.

Transform Tree

Frames are organized in a tree structure where each frame has exactly one parent, preventing loops.

Translation

Position offset between frames in X, Y, Z coordinates (typically in meters).

Rotation

Orientation difference expressed as Roll, Pitch, Yaw angles or quaternions.

Static vs Dynamic

Static transforms are fixed (sensor mounts). Dynamic transforms change over time (joint movements).

Frame Hierarchy

Common pattern: map → odom → base_link → sensor frames. Each level adds specific transform information.
